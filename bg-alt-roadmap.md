- [ ] How to install postgresql
  - [ ] using pacakge managers (apt, yum)
  - [ ] using docker

  - [ ] managing postgres server `pg_ctl`

  - [ ] Basic data types

 - [ ] Infrastructure DBA:
    - [ ] Replication: streaming replication, logical replication

    - [ ] Monitoring (prometheus .etc)
